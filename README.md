# GEO Analyser

**Generative Engine Optimization Desktop Application**

Track how your website appears in AI-generated responses from ChatGPT, Claude, Gemini, and more.

## ğŸ“¦ Installation

### Download Pre-built

Download the latest release for your platform:
- **Windows**: `GEO-Analyser-1.0.0.exe`
- **macOS**: `GEO-Analyser-1.0.0.dmg`

### Build from Source

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/geo-analyser.git
cd geo-analyser

# Install dependencies
npm install

# Run in development
npm run electron:dev

# Build for production
npm run build:win  # Windows
npm run build:mac  # macOS
```
